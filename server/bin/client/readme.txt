客户端测试程序使用说明：

1、该目录下的所有动态库（.so文件）均为AnyChat SDK运行时所需加载的库文件；

2、helloAnyChat_c++ 应用程序由src\client\c++\helloAnyChat目录下的源代码编译生成；

3、helloAnyChat_qt_x86 应用程序由src\client\Qt5\helloAnyChat目录下的源代码编译生成(32bit)；

4、./runhelloAnyChat.sh	可启动helloAnyChat_c++应用程序；

3、./starthelloAnyChatQt.sh	可启动helloAnyChat_qt应用程序(http://bbs.anychat.cn/forum.php?mod=viewthread&tid=1034&extra=page%3D1)
