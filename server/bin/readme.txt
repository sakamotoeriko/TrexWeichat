client		AnyChat客户端SDK组件，需要随用户客户端程序一同发布，并注册相关的组件
		参考：http://www.anychat.cn/faq/index.php?action=artikel&cat=7&id=242&artlang=zh

server		AnyChat服务器运行程序，需要部署在服务器上
		参考：http://bbs.anychat.cn/forum.php?mod=viewthread&tid=90&extra=page%3D1

serversdk	AnyChat服务器SDK组件，可以用来开发业务层服务器，实现AnyChat的功能扩展
		参考：http://www.anychat.cn/faq/index.php?action=artikel&cat=1&id=240

demo		AnyChat SDK客户端演示程序（由示例代码编译）

recordserver	AnyChat中心录像服务器，可以在服务器端进行视频录制，包括将通话双方的视频录制到同一个视频文件中
		参考：http://bbs.anychat.cn/forum.php?mod=viewthread&tid=20&extra=page%3D1
