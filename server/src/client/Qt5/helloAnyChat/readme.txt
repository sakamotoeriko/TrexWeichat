代码说明：

1、开发环境为QT5.3（linux or windows）

2、功能：

   2.1 启动软件（自动初始化并登录服务器）

   2.2 进入房间（用户可填入房间号码）

   2.3 双击选择对应用户列表中的用户，自动打开远程用户视频（列表会实时刷新）

   2.4 实现远程视频开始后，可以再输入框输入文字，进行文字聊天

   2.5 离开房间（离开后可重新进入任意房间）

   2.6 关闭程序（自动退出服务器并释放资源）

3、重要说明

   3.1 本例子视频使用回调方式，并由调用者进行视频绘制，没有使用内核显示模式；

   3.2 本例子音频使用SDK内核自动播放模式，并没有使用回调方式获取音频数据；

   3.3 本程序为使开发者快速入门，因此功能不会太多，仅实现本地音视频和同时请求一个用户音视频进行聊天，还有文字聊天（需请求视频成功）；
